# Generate Report

## Overview

Generate comprehensive reports summarizing all validation results including completeness checks, code quality, and overall repository health. Reports provide actionable insights for improvement.

[../plan.md](../plan.md)

## Inputs

- Required files check results
- Code quality metrics
- All validation data

## Outputs

- Comprehensive HTML report
- Text summary for console
- JSON data for tooling
- Overall health score

## Steps

1. Aggregate all validation results
2. Calculate overall health score
3. Generate reports in multiple formats
4. Highlight top priority issues

- [ ] Health score is meaningful
- [ ] Multiple output formats supported
- [ ] Reports guide improvements
