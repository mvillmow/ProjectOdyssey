# Tutorial Writer

## Overview
Create a prompt template for writing comprehensive tutorials. The template guides users through step-by-step instructions with clear explanations and working examples.

## Parent Plan
[../plan.md](../plan.md)

## Child Plans
None (implementation level)

## Inputs
- Tutorial topic and learning objectives
- Target audience understanding
- Code examples and demonstrations
- Tutorial writing standards

## Outputs
- Tutorial writer prompt template
- Tutorial structure guidelines
- Structured output format
- Few-shot examples
- Complete tutorial content

## Steps
1. Design tutorial structure with learning flow
2. Create prompt with teaching guidelines
3. Add few-shot tutorial examples
4. Define code example integration

## Success Criteria
- [ ] Template creates clear tutorials
- [ ] Learning objectives are stated
- [ ] Steps are logical and sequential
- [ ] Examples are working and relevant
- [ ] Explanations are clear
- [ ] Template adapts to audience level

## Notes
Include sections: introduction with objectives, prerequisites, step-by-step instructions, code examples with explanations, common issues and solutions, next steps. Use XML tags like <tutorial>, <objective>, <step>, <example>. Keep explanations clear and beginner-friendly.
