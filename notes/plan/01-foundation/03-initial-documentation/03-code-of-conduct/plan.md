# Code of Conduct

## Overview
Create a CODE_OF_CONDUCT.md file that establishes community guidelines and expectations for behavior. This ensures a welcoming, inclusive, and respectful environment for all contributors.

A code of conduct is essential for building a healthy community. Use a well-established template and customize minimally. Make it clear, enforceable, and welcoming.
